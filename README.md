# CryptoCritters

**Your Portfolio Has Paws.** 🐾

A gamified BSC portfolio tracker where tokens become AI-generated critters. Health, happiness, and evolution depend on real market data + your DeFi actions.

## Features

- 🔗 **Wallet Connect** - Scan BSC wallet to generate critters
- 🎨 **AI-Generated Critters** - Unique personalities based on token data
- 📊 **Real-Time Stats** - Health (price stability), Happiness (hold time), Energy (liquidity)
- 🎮 **Interact** - Feed, Play, Clean, Train your critters
- ✨ **Evolution System** - 30-day holds lead to rare NFT badges
- 🧠 **Educational** - Learn DeFi through gameplay

## Tech Stack

- **Backend**: Node.js, Express, TypeScript, ethers.js v6
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, Framer Motion
- **AI**: HuggingFace for avatar generation
- **Chain**: BNB Smart Chain

## Quick Start

```bash
# Install dependencies
cd backend && npm install
cd ../frontend && npm install

# Start backend
cd backend && npm run dev

# Start frontend (new terminal)
cd frontend && npm run dev
```
